# Study Materials

A personal collection of study notes, code samples, and reference material covering computer science, low-level programming, and related tech topics.

## About

This repository is my long-term home for the stuff I learn and want to come back to. Instead of scattering notes across random folders, screenshots, and bookmarks, everything lives here in one place — versioned, searchable, and easy to share.

Expect a mix of:

- **Computer Science fundamentals** — data structures, algorithms, operating systems, networking, databases.
- **Systems & tooling** — Linux internals, compilers, build systems, debuggers.
- **Misc tech notes** — anything else worth remembering: papers I've read, talks I've watched, problems I've solved.

## Goals

- Build a strong, written foundation across CS topics.
- Keep notes in plain text / Markdown so they stay portable.
- Pair theory with runnable code whenever possible.
- Make it easy for future-me (and anyone else) to learn from.


Folders will be added as content grows — not every directory exists yet.

## How I Use It

- New topic → create a folder, drop in a `README.md` with notes.
- Code example → keep it self-contained and runnable.
- Reference → link the source so I can revisit the original.
- Review → revisit older notes and refine them over time.
